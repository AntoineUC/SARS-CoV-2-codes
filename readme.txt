This folder contains:

1) Datasets S1 and S2 and the code used for their analysis, see the file « sse.R »

2) Database S3, see the file « traceDatSaved.Rdata », and the code used for its analysis, see « sse india.R »

3) Database S4, see the file « sse_korea_2020.txt », and the code used for its analysis, see « sse korea 2020.R »

4) Database S5, see the file « sse_korea_2021_period_2.txt », and the code used for its analysis, see « sse korea 2021.R »

5) Databases S6 and S7, see the files « colorado.txt » (Colorado data) and « clusters.Rdata » (all data except Colorado), and the codes used for their analysis, see « fit cluster XXX.R »

6) The codes used for robustness checks, see « robustness_XXX.R » (robustness due to missing data) and « robustness_XXX_bis.R » (robustness due to poor recording)

7) The code used to produce Figure 1, called « hillplot_burr_fig1.R »
